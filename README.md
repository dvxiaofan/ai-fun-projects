# AI Fun Projects

用 AI 编程快速实现的创意小工具 / 小游戏合集。

每个子目录是一个独立项目，纯前端为主，浏览器打开 `index.html` 即可运行，无需安装依赖。

---

## 技术栈

- **主体**：HTML + CSS + JavaScript（原生，零依赖）
- **部分项目**：Vite + TypeScript（较复杂的项目）
- **画布类**：Canvas 2D / WebGL
- **音频类**：Web Audio API

## 使用方式

```bash
# 进入任意子目录
cd snake

# 直接浏览器打开
open index.html

# 或者起个本地服务（需要时）
python3 -m http.server 8080
```

## 开发理念

- **快速出活**：一个项目尽量 30 分钟内能跑起来
- **零门槛**：打开 HTML 就能玩，不用装环境
- **可魔改**：代码结构清晰，方便二开和魔改
- **独立运行**：每个项目自包含，不依赖外部服务
